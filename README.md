# Pizza Sales SQL Project

## Overview
Analyzed pizza sales data using SQL to understand customer ordering behavior, revenue trends, top-selling pizzas, and overall business performance.

## SQL Concepts Used
- Joins
- Aggregate Functions
- Subqueries
- CTE
- Views
- Window Functions
- CASE Statements
- Date & Time Functions

## Key Analysis
- Total revenue generation
- Top-selling pizzas
- Most common pizza size
- Busiest order hours
- Running total of revenue
- Highest revenue-generating pizza type

## Tools Used
- MySQL
- MySQL Workbench
- SQL

## Outcome
Converted raw transactional data into meaningful business insights and strengthened SQL problem-solving skills.
